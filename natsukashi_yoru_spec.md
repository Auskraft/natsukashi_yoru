# Natsukashi Yoru — Project Spec

## Обзор
Игровой монорепо на Flutter + Flame. Несколько классических игр с общей архитектурой, единой темой и системой ачивок (в будущем).

- **Репо:** https://github.com/Auskraft/natsukashi_yoru (публичное)
- **Organization:** com.auskraft
- **Flutter версия:** 3.41.7 (stable)
- **Android language:** Kotlin
- **Платформа:** Android only (apk)

---

## Архитектура
**Feature-Based + Component-Oriented**

```
lib/
├── core/
│   ├── theme/          # цвета, шрифты, реколор
│   ├── components/     # общие виджеты (кнопки, счёт, пауза, HUD)
│   ├── audio/          # звуки
│   └── achievements/   # 🔒 заглушка — реализуем позже
├── features/
│   ├── menu/           # главное меню выбора игры
│   ├── snake/
│   ├── tetris/
│   ├── match3/
│   ├── bejeweled/
│   ├── puyo_puyo/
│   └── minesweeper/
└── main.dart
```

### Структура каждой фичи
```
<game_name>/
├── components/     # игровые объекты (кусочки, блоки, шарики и т.д.)
├── game/           # логика Flame (GameLoop, обработка ввода)
├── ui/             # HUD, экраны паузы, game over
└── <game_name>_game.dart  # точка входа фичи
```

## Игры (6 штук)

| Игра | Описание | Статус |
|------|----------|--------|
| **Snake** | Классическая змейка | ✅ играбельна (juice + рекорды) |
| **Tetris** | Падающие блоки | ✅ играбельна (juice + рекорды) |
| **Match3** | 3 в ряд (базовая механика) | ✅ играбельна (блиц 60с, juice + рекорды) |
| **Bejeweled** | Обмен соседних камней, матч по цвету | ⬜ заглушка |
| **Puyo Puyo** | Цветные шарики, combo-цепочки | ⬜ заглушка |
| **Minesweeper** | Сапёр | ⬜ заглушка |

## Задачи для Claude Code

1. Создать структуру папок согласно архитектуре
2. Настроить `pubspec.yaml` с Flame и зависимостями
3. Создать `main.dart` с роутингом на меню
4. Создать заглушки для каждой фичи (пустые `_game.dart` с базовым классом)
5. Создать `core/theme/` с базовой темой и поддержкой реколора
6. Создать `core/achievements/README.md` как напоминание
7. Первый рабочий запуск — меню с 6 кнопками игр
