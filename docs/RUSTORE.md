## Сборка релиза

Подпись релиза уже настроена в `android/app/build.gradle.kts`: параметры берутся
из `android/key.properties` (если файла нет — собирается debug-ключом).

1. Сгенерировать хранилище ключей (один раз):
   ```bash
   keytool -genkey -v -keystore android/app/upload-keystore.jks \
     -keyalg RSA -keysize 2048 -validity 10000 -alias upload
   ```
   ⚠️ Сохраните `.jks` и пароли надёжно — без них нельзя обновлять приложение.
2. Создать `android/key.properties` из `android/key.properties.example` и
   заполнить. ⚠️ Ключи и `key.properties` в git НЕ попадают (см. `.gitignore`).
3. Собрать:
   ```bash
   flutter build appbundle --release   # AAB (предпочтительно)
   # или
   flutter build apk --release         # APK
   ```
4. Загрузить артефакт в консоль RuStore, заполнить карточку (описание, категория,
   возраст, скриншоты), отправить на модерацию.
