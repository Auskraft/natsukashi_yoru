# 🔒 achievements — заглушка

Система ачивок **не реализуется в первой итерации** — это напоминание и архитектурный задел.

## Что планируется

- Реестр ачивок (id, название, описание, иконка, условие).
- Хранилище разблокированных ачивок через `shared_preferences`.
- Хуки из игр: каждая фича сообщает о событиях (счёт, комбо, победа),
  менеджер ачивок проверяет условия и разблокирует.
- Экран просмотра ачивок (из главного меню).

## Где подключать

- Игры (`features/<game>/`) эмитят игровые события.
- Менеджер ачивок (здесь) подписывается на события и пишет прогресс.
- UI ачивок — отдельный экран, вход из `features/menu/`.

> Не удалять папку: служит маркером запланированной фичи.
